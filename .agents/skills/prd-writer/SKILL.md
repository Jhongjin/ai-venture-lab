# PRD Writer

Start from `templates/PRD.md` and `templates/MVP_SPEC.md`.

Keep the MVP narrow:

- One core user
- One painful workflow
- One measurable success metric
- One launchable prototype path

Always include non-goals and privacy/security notes.
